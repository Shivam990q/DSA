## 🎯 WHAT THIS IS

A **complete, ground-up mastery path for Java** — from "what is a compiler?" to "tuning the G1 garbage collector and writing lock-free concurrent code." Designed for **one learner going from zero to grandmaster**, leaving *nothing* out.

Every topic is taught the same way — the **4-layer method**:

1. **🧠 Theory** — what it is, why it exists, how it works under the hood, all terminology defined.
2. **⌨️ Practical** — runnable code examples you type and break.
3. **❓ Questions** — self-test + interview questions with answers.
4. **⚠️ Pitfalls** — the mistakes everyone makes, so you don't.

| Phase | Theme | You'll be able to... |
