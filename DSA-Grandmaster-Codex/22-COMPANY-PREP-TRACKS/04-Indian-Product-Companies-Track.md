# 🇮🇳 Indian Product Companies Track

> Flipkart, Swiggy, Zomato, Zoho, Freshworks, Paytm, PhonePe, MakeMyTrip, Ola, Myntra, Nykaa, Meesho, CRED, Groww, Dream11, etc.

Strong DSA bar (close to FAANG-lite), with some CS subjects + design for senior roles. Excellent compensation and growth.

---

## THE HIRING FUNNEL
```
1. Online Assessment (DSA — HackerRank/HackerEarth/own platform)
2. 1-3 technical rounds (DSA Medium-Hard)
3. Machine coding / LLD round (some — e.g., build a working mini-system in 1-2 hrs)
4. System design (senior roles)
5. Hiring manager + HR
```

---

## WHAT THEY TEST
- **DSA** ⭐ — [LeetCode](https://leetcode.com) Medium-Hard (the core)
- **Machine coding / LLD** (some) — design + implement a working system (e.g., Splitwise, parking lot) in 90-120 min
- **System design** (senior)
- **CS fundamentals** (some — OS/DBMS/CN basics)
- **Projects + resume**

## THE MACHINE CODING ROUND ⭐ (unique to many Indian product cos)
You're asked to build a **working mini-application** in 90-120 minutes:
- Examples: Splitwise, parking lot, snake & ladder, elevator, in-memory cache, rate limiter
- Evaluated on: working code, OOP design, SOLID principles, extensibility, clean code
- **Prepare**: practice LLD problems; know design patterns; write clean, modular, tested code fast

Resources: search "machine coding round questions", "low level design interview"; practice on paper + IDE.
