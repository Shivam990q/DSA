# 🦄 Unicorns & Product Startups Track

> Stripe, Razorpay, Atlassian, Databricks, Coinbase, Robinhood, Plaid, Notion, Figma, Canva, Confluent, HashiCorp, GitLab, and high-growth scale-ups.

High DSA bar + heavy emphasis on **practical engineering** (real-world coding, debugging, building).

---

## THE HIRING FUNNEL
```
1. Resume / referral / OA
2. Phone screen (coding — often practical, not pure puzzle)
3. Onsite loop:
   - 1-2 DSA / practical coding
   - 1 "real-world" coding (debug/extend a codebase, build a feature)
   - 1 system design
   - 1 behavioral / values
4. Offer
```

---

## WHAT MAKES THEM DIFFERENT
- **Practical coding** ⭐: Stripe is famous for "integration"-style problems — build/debug working code, handle edge cases, work with APIs (less pure-puzzle, more real engineering)
- **High code quality bar**: clean, tested, production-minded code
- **System design** matters early
- **Culture/values fit** is taken seriously

---

## COMPANY FLAVORS
| Company | Notes |
|---------|-------|
| **Stripe** | Practical, multi-part problems; build real features; edge cases; API design ⭐ |
| **Razorpay** | DSA + machine coding + design (fintech scale) |

| **Databricks** | High DSA + distributed systems depth |

---

## THE PREP PLAN (4-6 months)
```
Month 1-2: DSA foundations + patterns
Month 3-4: LeetCode Mediums + PRACTICAL coding practice
Month 5:   System design + build a real project (shows engineering)
Month 6:   Mocks + company-specific + apply
```

### Practical coding prep (the differentiator)
- Build small full features (not just algorithm puzzles)
- Practice debugging existing code
- Work with APIs, handle errors and edge cases
- Write clean, tested code (production mindset)
- For Stripe specifically: practice multi-stage problems where each stage builds on the last

---

## THE RESOURCE STACK
- **DSA**: [NeetCode](https://neetcode.io) 150, [LeetCode](https://leetcode.com) company-tagged
- **Practical**: build real projects; contribute to open source
- **System design**: Alex Xu + [system-design-primer](https://github.com/donnemartin/system-design-primer)
- **For fintech**: understand payments, security basics (Coinbase/Stripe/Plaid)

---

## THE BAR
- LeetCode Medium fluently
- Can build clean, working, tested features
- System design competence
- Strong engineering judgment

---

## THE STRATEGY
- These companies value **engineers who ship**, not just puzzle-solvers.
- Your **projects + GitHub** matter more here than at pure-DSA shops.
- Practice the practical/integration style (especially for Stripe-tier).
- Culture fit is real — research and align with their values.
