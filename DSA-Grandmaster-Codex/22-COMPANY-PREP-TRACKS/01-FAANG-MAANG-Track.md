# 👑 FAANG / MAANG Track

> Google, Meta (Facebook), Amazon, Apple, Microsoft, Netflix — and the broader "MAANG" tier.

---

## THE HIRING FUNNEL
```
1. Resume screen / referral
2. Online Assessment (OA) — for some roles/campus (HackerRank/CodeSignal)
3. Phone screen — 1 coding problem (45 min)
4. Onsite / virtual loop — 4-6 rounds:
   - 2-4 coding (DSA)
   - 1 system design (L5+ / experienced)
   - 1 behavioral
5. Hiring committee + team match
6. Offer + negotiation
```

---

## WHAT THEY TEST
- **DSA** (the core): [LeetCode](https://leetcode.com) Medium-Hard, the 25 patterns
- **System design** (mid/senior): scalable architecture
- **Behavioral**: STAR stories, company values
- **Low-level design / OOP** (some): Amazon, etc.

---

## DSA BAR
- Solve Medium in <20 min, Hard in <40 min
- All 25 patterns at Stage-4 mastery
- Maps to [Level 5: FAANG DSA](../01-LEVELS-PROGRESSION/Level-05-FAANG-DSA.md)

## THE 6-MONTH PREP PLAN
```
Month 1-2: DSA foundations + 25 patterns (NeetCode 150 / Striver)
Month 3-4: LeetCode Mediums + company-tagged (LC Premium); start mocks
Month 5:   Hards + system design + behavioral stories (10 STAR)
Month 6:   Mocks (5/week) + full-day simulations + apply
```

## THE EXIT CRITERIA (you're ready when)
- 70%+ of LC Hards solved in <45 min
- All 25 patterns fluent
- 10 polished behavioral stories
- Can design Twitter/rate-limiter/URL-shortener
- Pass 4/5 mock rounds at "would hire"

---

## NEGOTIATION
- Use [levels.fyi](https://www.levels.fyi) for compensation data
- Never accept on the spot; get competing offers
- Negotiate total comp, sign-on, equity, start date
- See [`../13-INTERVIEW-DSA-UNIVERSE/11-COMPENDIUM-Interview-Mastery.md`](../13-INTERVIEW-DSA-UNIVERSE/11-COMPENDIUM-Interview-Mastery.md)

## THE MINDSET
- Most people fail FAANG interviews multiple times before passing. Normal.
- Apply when ~80% ready (you learn from real interviews).
- Keep practicing DURING interview loops.
