# 🔄 The Universal Hiring Funnel

> *"Every company hires through the same funnel. Understand it, optimize each stage."*

---

## THE FUNNEL (every company, roughly)
```
APPLICATION → SCREEN → ASSESSMENT → INTERVIEWS → DECISION → OFFER → NEGOTIATION
     │           │          │            │           │         │          │
   resume    recruiter    OA/test    technical+   hiring     offer    comp + terms
   +referral  +ATS                   behavioral   committee  letter
```

---

## STAGE 1: APPLICATION
- **Goal**: get past the resume screen
- **Optimize**: tailored resume, referrals, apply early, strong profile
- **Conversion**: low (most applications get no response — normal)
- See [`10-Resume-And-Profile-Building.md`](./10-Resume-And-Profile-Building.md)

## STAGE 2: RECRUITER SCREEN
- **Goal**: confirm basic fit, role interest, logistics
- **Optimize**: clear communication, know the role, have questions ready
- Quick call (15-30 min); mostly behavioral/logistical

## STAGE 3: ONLINE ASSESSMENT (OA)
- **Goal**: pass the automated coding/aptitude screen
- **Platforms**: [HackerRank](https://www.hackerrank.com), [CodeSignal](https://codesignal.com), [Codility](https://www.codility.com), [HackerEarth](https://www.hackerearth.com) (see [`../20-RESOURCES-AND-REFERENCES/14-Online-Judges-And-Platforms.md`](../20-RESOURCES-AND-REFERENCES/14-Online-Judges-And-Platforms.md))
- **Optimize**: practice on the SAME platform; know the format; handle edge cases

## STAGE 4: INTERVIEWS
- **Goal**: demonstrate skill + communication + fit
- **Types**: coding (DSA), system design, behavioral, domain
- **Optimize**: mocks, narration, the [problem-attack protocol](../03-PROBLEM-SOLVING-FOUNDATIONS/12-The-Problem-Attack-Protocol.md)

## STAGE 5: DECISION (hiring committee / debrief)
- Interviewers submit feedback; committee/manager decides
- **You can't control this**, but strong, consistent signals across rounds help

## STAGE 6: OFFER
- The offer letter (base, equity, bonus, sign-on, level)

## STAGE 7: NEGOTIATION
- **Always negotiate** (politely, with data)
- [levels.fyi](https://www.levels.fyi) for comp data
- Get competing offers
- Negotiate total comp, not just base

---

## CONVERSION RATES (rough reality)
- Applications → screen: ~5-20% (varies hugely)
- Screen → OA/interview: ~50%
- OA → onsite: ~30-50%
- Onsite → offer: ~20-40%

**Implication**: you need MANY applications to land offers. It's a funnel — widen the top (apply broadly), optimize each stage.

---

## OPTIMIZING THE WHOLE FUNNEL
1. **Top of funnel**: apply to 50-100+ companies, get referrals
2. **Middle**: ace OAs (practice the platform), prep CS subjects
3. **Interviews**: mocks + narration + the protocol
4. **Offer**: negotiate

---

## THE TIMELINE
- Full cycle per company: 2-8 weeks
- Job search overall: 2-6 months of active effort
- Run multiple companies in PARALLEL (don't do them one at a time)

---

## THE PARALLEL STRATEGY
- Apply to many companies at once
- Schedule interviews to cluster (so you're "warmed up")
- Use earlier interviews as practice for dream companies
- Time offers to create leverage (negotiation)

---

## THE EMOTIONAL REALITY
- Rejection is the norm, not failure
- Ghosting is common (companies don't always respond)
- One offer is enough — but you need many shots
- Keep practicing DURING the search
- Persistence + skill + numbers = success

---

## THE FINAL TRUTH
> **"The funnel is the same for everyone. The grandmaster simply:  
>  applies broadly, prepares deeply, communicates clearly, negotiates confidently,  
>  and never quits the search."**
