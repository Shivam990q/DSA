## 🗺️ THE COMPANY TIER MAP

| Tier | Examples | What they test | DSA bar |
|------|----------|----------------|---------|
| **FAANG/MAANG** | Google, Meta, Amazon, Apple, MS, Netflix | DSA + system design + behavioral | Very High |
| **Global Big Tech** | Adobe, Salesforce, Uber, Nvidia, LinkedIn | DSA + design + domain | High |
| **Top Unicorns** | Stripe, Atlassian, Razorpay, Databricks | DSA + design + practical | High |
| **Indian Product** | Flipkart, Swiggy, Zomato, Zoho, Freshworks | DSA + some CS subjects + design | Medium-High |
| **Quant/HFT** | Jane Street, Optiver, Tower, Hudson River | Math + DSA + probability + low-latency | Extreme (different) |
| **Service** | TCS, Infosys, Wipro, Accenture | Aptitude + CS subjects + basic DSA | Low-Medium |
| **Startups** | Seed/Series-A | Practical coding + projects + culture | Variable |

## 🎯 HOW TO USE THIS UNIVERSE
1. Identify your **target tier(s)** (be ambitious but realistic — apply across tiers).
2. Read that track's file for the **exact prep plan**.
3. Cross-reference with the [Levels](../01-LEVELS-PROGRESSION/00-Index.md) (e.g., FAANG = Level 5).
4. Use the [Resources](../20-RESOURCES-AND-REFERENCES/00-Index.md) and [Sheets](../20-RESOURCES-AND-REFERENCES/10-Curated-Sheets/00-Index.md) referenced in each track.
5. Build your [resume](./10-Resume-And-Profile-Building.md) and [apply](./09-Off-Campus-And-Referrals-Track.md).

## 🪔 THE UNIVERSAL TRUTH
No matter the company, the formula is:
> **Strong fundamentals (DSA + CS subjects) + targeted practice (company patterns) + communication (interviews) + a profile that gets you the interview (resume + projects + referrals).**

Master these four, and every door opens.
