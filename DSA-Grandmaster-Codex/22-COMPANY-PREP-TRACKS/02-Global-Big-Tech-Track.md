# 🌍 Global Big Tech Track

> Adobe, Salesforce, Nvidia, Uber, Airbnb, LinkedIn, Oracle, SAP, VMware, Atlassian, Snowflake, Databricks, Twitter/X, Spotify, PayPal, Visa, Mastercard, Cisco, Intuit, ServiceNow, Workday, etc.

The tier just below/beside FAANG. Similar process, excellent compensation, often slightly more accessible.

---

## THE HIRING FUNNEL
```
1. Resume / referral / OA (CodeSignal/HackerRank)
2. Phone screen (1 coding problem)
3. Onsite loop — 4-5 rounds:
   - 2-3 coding (DSA)
   - 1 system design (mid/senior)
   - 1 behavioral / hiring manager
4. Offer + negotiation
```

---

## WHAT THEY TEST
- **DSA**: [LeetCode](https://leetcode.com) Medium-Hard (similar to FAANG, sometimes slightly easier)
- **System design**: for mid/senior
- **Domain knowledge**: e.g., Nvidia (systems/CUDA for some roles), Visa/Mastercard (fintech/security), Databricks/Snowflake (data/distributed)
- **Behavioral**

---

## COMPANY FLAVORS
| Company | Notes |
|---------|-------|
| **Adobe** | Strong DSA + some domain (graphics/product); moderate bar |
| **Salesforce** | DSA + system design + Apex/platform for some roles |
| **Uber/Airbnb/LinkedIn** | FAANG-tier DSA + design |
| **Nvidia** | DSA + systems/C++/CUDA for many roles |
| **Oracle/SAP/VMware** | DSA + systems/database depth |
| **Databricks/Snowflake** | High DSA + distributed systems |
| **Visa/Mastercard/PayPal** | DSA + security/fintech + system design |
| **Atlassian** | DSA + practical + design |

---

## THE PREP PLAN (4-6 months)
Essentially the [FAANG track](./01-FAANG-MAANG-Track.md) plan, with added domain study for specialized roles:
```
Month 1-2: DSA foundations + patterns
Month 3-4: LeetCode Mediums + company-tagged + start design
Month 5:   Hards + system design + domain (if applicable) + behavioral
Month 6:   Mocks + apply
```

---

## THE RESOURCE STACK
- Same as FAANG: [NeetCode](https://neetcode.io) 150, Blind 75, LC company-tagged
- System design: Alex Xu + [ByteByteGo](https://bytebytego.com) + [system-design-primer](https://github.com/donnemartin/system-design-primer)
- **Domain-specific** (as needed): CUDA/C++ (Nvidia), distributed systems (Databricks/Snowflake), security (Visa/Mastercard)

---

## THE BAR
- LeetCode Medium fluently, decent Hards
- System design for mid/senior
- Slightly more forgiving than top FAANG, but still high

---

## THE STRATEGY
- These are excellent targets if FAANG feels like a stretch — same skills, slightly higher hit rate.
- Apply broadly across this tier AND FAANG simultaneously (same prep).
- Research the specific role's domain needs (a Nvidia systems role ≠ a generic web role).
