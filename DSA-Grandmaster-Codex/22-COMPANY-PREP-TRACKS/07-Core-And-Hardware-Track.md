# 🔧 Core & Hardware / Systems Track

> Qualcomm, Intel, Nvidia (systems), AMD, ARM, Texas Instruments, Samsung, Micron, Broadcom, Cisco (systems), and embedded/firmware/VLSI roles.

For those who love low-level systems, C/C++, hardware, and embedded. Different emphasis: systems knowledge + C + domain over [LeetCode](https://leetcode.com) grinding.

---

## THE HIRING FUNNEL
```
1. Resume / OA (C/C++ + aptitude + domain MCQs)
2. Technical rounds:
   - C/C++ deep (pointers, memory, bit manipulation)
   - OS / computer architecture / digital concepts
   - Domain (embedded, VLSI, networking, etc.)
   - DSA (moderate)
3. HR
```

---

## WHAT THEY TEST
1. **C / C++ mastery** ⭐ — pointers, memory management, bit manipulation, structs, unions, volatile, const
2. **Computer Organization & Architecture** ⭐ — pipelining, cache, memory hierarchy, instruction sets
3. **Operating Systems** — processes, scheduling, memory, interrupts
4. **Digital Electronics** (hardware roles) — logic gates, flip-flops, FSMs
5. **Domain**: embedded (microcontrollers, RTOS), VLSI (Verilog/VHDL), networking (protocols)
6. **DSA** — moderate (arrays, linked lists, trees; embedded-relevant)
7. **Bit manipulation** ⭐ — heavily used in embedded

---

## ROLE TYPES
- **Embedded / Firmware** — C, microcontrollers, RTOS, hardware interfacing
- **VLSI / Chip Design** — Verilog/VHDL, digital design, EDA tools
- **Systems Software** — OS, drivers, low-level C/C++
- **Networking** — protocols, packet processing

---

## THE PREP PLAN (3-5 months)
```
Month 1: C/C++ deep (pointers, memory, bit tricks) ⭐
Month 2: Computer Architecture + OS
Month 3: Domain (embedded/VLSI/networking — pick your path)
Month 4: DSA (moderate) + bit manipulation
Month 5: Domain projects + interviews
```

- Solid CO/architecture + OS

- Moderate DSA

## THE STRATEGY
- This track rewards **systems depth** over LeetCode volume.
- **C/C++ mastery** is the #1 differentiator (many candidates are weak here).
- Build **domain projects** (an embedded project, an FPGA design) — they matter enormously.
- GATE (India) preparation overlaps heavily with this track's CS subjects.
