# 🏢 Service Companies Track

> TCS, Infosys, Wipro, Cognizant, Accenture, Capgemini, HCL, Tech Mahindra, LTIMindtree, etc.

These are the **highest-volume recruiters** (especially in India). Easier DSA bar, but **aptitude + CS subjects + communication** matter a lot. Great entry point + stable career start.

---

## THE HIRING FUNNEL
```
1. Online test (aptitude + reasoning + verbal + basic coding/pseudocode)
2. Technical interview (CS subjects + basic DSA + projects)
3. HR interview (communication, fit, relocation, salary)
```

---

## WHAT THEY TEST (in order of weight)
1. **Aptitude & Reasoning** (the filter) ⭐ — quant, logical, verbal
2. **CS Core Subjects** ⭐ — OS, DBMS, CN, OOP (interview)
3. **Basic DSA / Coding** — arrays, strings, sorting, basic problems (Easy-Medium)
4. **Projects** — be able to explain your resume projects
5. **Communication / HR** — confidence, clarity, fit

---

## COMPANY-SPECIFIC TESTS
| Company    | Test name        | Key sections                          |
|------------|------------------|---------------------------------------|
| TCS        | **TCS NQT**      | Numerical, verbal, reasoning + coding |
| Infosys    | **InfyTQ**       | Aptitude + pseudocode + Python/Java + coding |
| Wipro      | **Elite NLTH**   | Aptitude + essay writing + coding     |
| Cognizant  | **GenC / GenC Next** | Aptitude + coding                 |
| Accenture  | Assessment       | Cognitive + technical + coding + comms|
| Capgemini  | **Exceller / game-based** | Pseudocode + aptitude + essay |

---

## THE PREP PLAN (6-8 weeks)
```
Weeks 1-3: APTITUDE ⭐ (the filter)
  - Quant (R.S. Aggarwal / IndiaBix / PrepInsta)
  - Reasoning + Verbal
  - Company-specific patterns (PrepInsta / Faceprep)
  - Timed mock tests

Weeks 4-5: CS CORE SUBJECTS ⭐
  - OS, DBMS, CN, OOP (Gate Smashers + GfG Last Minute Notes)
  - SQL basics

Weeks 6-7: BASIC DSA + CODING
  - Arrays, strings, sorting, searching, basic recursion
  - Pseudocode practice (for Infosys/Capgemini)
  - 50-100 Easy + some Medium problems

Week 8: PROJECTS + HR + mock interviews
  - Polish 1-2 resume projects
  - Common HR questions
  - Communication practice
```

## HR INTERVIEW — COMMON QUESTIONS
- Tell me about yourself
- Why this company?
- Strengths and weaknesses
- Are you willing to relocate?
- Where do you see yourself in 5 years?
- Why should we hire you?

Prepare crisp, honest, confident answers.

---

## THE STRATEGY
- Service companies are a **great first job** + stable start; many people later switch to product companies after 1-2 years.
- The aptitude round filters most people — **don't get filtered there**.
- DSA bar is low — basic competence + good communication wins.
- You can prep for service companies in **6-8 weeks** alongside deeper DSA for product companies.

---

## THE UPGRADE PATH
Many start at a service company, then switch to a product company/unicorn after 1-2 years by continuing DSA prep. Keep practicing even after landing the service role.
