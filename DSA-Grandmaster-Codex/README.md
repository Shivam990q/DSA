## ⚔️ HOW TO USE THIS CODEX

### Path 1: The Linear Path (Recommended for Beginners)
Walk Levels 0 → 10 sequentially. Each level builds on the prior. No skipping. ~12-24 months for serious students.

### Path 2: The Domain Path (For Existing Programmers)
Pick a universe. Master it deeply. Move to the next. Best for engineers who want depth in DP, Graphs, Strings, etc.

### Path 3: The Goal Path (For Targeted Outcomes)
- 🎯 **FAANG in 3 months** → Level 0/1 (skim) → Level 5 (deep) → Universe 13
- 🎯 **CF Expert in 6 months** → Level 0-3 → Universe 12 (CF track)
- 🎯 **ICPC Regionalist** → Level 0-6 → Universe 12 (ICPC) + Universe 6, 7, 8
- 🎯 **Algorithm Researcher** → Levels 0-8 → Universe 13 (Algorithm Design Science)

### Path 4: The Grandmaster Path (Full Civilization)
Walk every level. Master every universe. Solve every problem. Attempt every simulation. Earn the title of **Codex Grandmaster**.
