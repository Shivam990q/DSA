# 🥇 IOI Systems — Olympiad Training

> *"The IOI is the Olympics of high-school programming. Individual, deep, subtask-scored."*

---

## I. THE FORMAT
- Individual (high-school level)
- 2 days × 5 hours, 3 problems per day
- Each problem: 100 points, with SUBTASKS for partial credit

---

## II. THE SUBTASK SYSTEM (the IOI-defining skill)
Each problem has subtasks with reduced constraints:
- Subtask 1 (n ≤ 10): brute force
- Subtask 2 (n ≤ 1000): O(n²)
- Subtask 3 (n ≤ 10⁵): optimal

**Strategy**: maximize TOTAL points. A 40-point partial on each of 3 problems (120) beats one 100-point full solution.

---

## III. THE STRATEGY
```
Min 0-20:   Read all 3; identify all subtasks
Min 20-120: Grab easy subtasks of ALL THREE (secure baseline)
Min 120-280: Push hardest subtasks of the problems you understand best
Min 280-300: Verify; ensure subtask solutions still pass
```

---

## IV. TRAINING PATH (for aspiring olympians)
1. **Foundations**: [USACO](https://www.usaco.org) Bronze → Silver → Gold → Platinum
2. **[USACO Guide](https://usaco.guide)** — phenomenal structured curriculum
3. **[Codeforces](https://codeforces.com)** — for speed + breadth
4. **Past IOI problems** ([oi.uz](https://oi.uz), ioi archives)
5. **National olympiad camps** (country-specific)

---

## V. THE IOI TECHNIQUE SET
- Binary search on answer
- DP (interval, tree, game, bitmask)
- Greedy with proof
- Segment trees, BIT
- Graph algorithms
- Clever observations

---

## VI. NATIONAL OLYMPIAD PATH (e.g., India: IOITC, ZCO/INOI)
- ZCO (Zonal Computing Olympiad) → INOI (Indian National) → IOITC (training camp) → IOI team
- Similar structures exist per country.

---

## VII. RESOURCES
- **USACO Guide** ⭐ (usaco.guide)
- **USACO contests** (free, monthly, subtask-style)
- **oi.uz** (IOI + national OI archive)
- **Codeforces** (for speed training)
- **[CSES Problem Set](https://cses.fi/problemset/)** (foundational)

---

## VIII. THE MINDSET
IOI rewards DEEP thinking over speed (5 hours, 3 problems). Train observation, proof, and subtask-maximization. The careful, methodical solver wins.
