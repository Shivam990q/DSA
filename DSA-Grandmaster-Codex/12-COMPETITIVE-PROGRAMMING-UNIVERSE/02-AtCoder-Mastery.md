---

## II. THE CONTEST TYPES
- **ABC** (Beginner): A-G/Ex, 100 min — start here
- **ARC** (Regular): harder, observation-heavy
- **AGC** (Grand): legendary difficulty, pure insight

---

## III. WHY ATCODER IS SPECIAL
1. **Clean statements** — minimal ambiguity
2. **Observation-heavy** — trains the "aha" muscle
3. **Excellent educational content** (EDPC, ACL)
4. **Immediate feedback** — know if you passed

---

## IV. THE MUST-DO: EDUCATIONAL DP CONTEST (EDPC)
26 problems (A-Z) covering all foundational DP:
- Knapsack, LCS, LIS
- Interval DP, tree DP, bitmask DP
- Probability/expected DP
- DP on graphs

**Solve all 26 before claiming DP mastery.**

---

## V. THE AtCoder LIBRARY PRACTICE CONTEST (ACL)
Problems exercising the official [AtCoder Library](https://github.com/atcoder/ac-library): segment tree, lazy segtree, DSU, max flow, min-cost flow, FFT, etc. Great for learning advanced DS.

---

## VI. THE CLIMB
- Brown → Green: ABC A-D consistently
- Green → Cyan: ABC E
- Cyan → Blue: ABC F, ARC C-D
- Blue → Yellow: ARC E, AGC B-C
- Yellow+: AGC C-E

---

## VII. TOOLS
- **AtCoder Problems** (kenkoooo.com) — track progress, filter by difficulty, "recommendations"
- Virtual contests on past ABCs
- AtCoder editorials (excellent quality)

---

## VIII. THE OBSERVATION TRAINING
AtCoder problems (especially ARC/AGC) reward the ability to:
1. Compute small cases by hand
2. Spot the pattern
3. Prove it
4. Implement quickly

This is the highest-leverage CP skill. AtCoder trains it best.
