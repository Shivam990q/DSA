## THE WEEKLY CONTEST RHYTHM
```
Sat: AtCoder ABC + LeetCode Biweekly (if scheduled)
Sun: LeetCode Weekly + Codeforces (varies)
Weekday evenings: Codeforces rounds (varies by timezone)
```

Aim for 1-3 live contests/week + virtual contests to fill gaps.

---

## THE DISCIPLINE
- Participate LIVE when possible (rating pressure = growth)
- Virtual the ones you miss (no rating, same practice)
- UPSOLVE everything afterward (the real learning)
