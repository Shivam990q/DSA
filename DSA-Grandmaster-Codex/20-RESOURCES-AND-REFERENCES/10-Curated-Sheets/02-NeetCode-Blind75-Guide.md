# 📋 NeetCode & Blind 75 — Complete Guide

> The most efficient video-based interview prep ecosystem, built around PATTERNS.

---

## BLIND 75 — THE ORIGIN
- A list of **75 problems** originally posted on TeamBlind by a Meta engineer
- Designed as the "minimum high-yield set" covering the core patterns
- Helped thousands clear Google, Amazon, Microsoft, Meta interviews
- **Best for**: time-crunched prep; the essential foundation

### Blind 75 categories
Arrays & Hashing, Two Pointers, Sliding Window, Stack, Binary Search, Linked List, Trees, Tries, Heap/Priority Queue, Backtracking, Graphs, Advanced Graphs, 1-D DP, 2-D DP, Greedy, Intervals, Math & Geometry, Bit Manipulation

---

## NEETCODE 150 — THE EXPANSION
- **150 problems**: Blind 75 + 75 more for fuller coverage
- Organized into the same pattern categories
- Each has a **video walkthrough** + clean Python/Java/C++/JS code
- **Best for**: thorough interview prep with guided learning

## NEETCODE 250 — THE COMPLETE SET
- Further extended to **250 problems** for deeper mastery
- Adds more variety per pattern

---

## THE NEETCODE ROADMAP ⭐
A visual **dependency graph** showing which topics unlock which. Start at Arrays & Hashing, follow the arrows. This prevents the "what do I learn next?" paralysis.

Available free at [neetcode.io](https://neetcode.io).

---

## HOW TO USE NEETCODE

```
Week 1-2:  Arrays & Hashing, Two Pointers, Sliding Window, Stack
Week 3-4:  Binary Search, Linked List, Trees
Week 5-6:  Tries, Heap, Backtracking
Week 7-8:  Graphs, Advanced Graphs
Week 9-10: 1-D DP, 2-D DP
Week 11:   Greedy, Intervals
Week 12:   Math & Geometry, Bit Manipulation
```

Per problem:
1. Attempt 20-30 min
2. If stuck, watch NeetCode's explanation for the IDEA
3. Code it yourself
4. Note the pattern
