# 📋 CP Ladders & CSES — Competitive Programming Paths

> For competitive programming, these structured ladders take you from beginner to grandmaster.

---

## CSES PROBLEM SET ⭐ (the structured CP curriculum)
- **~300 problems** at [cses.fi/problemset](https://cses.fi/problemset)
- **Sections**: Introductory Problems, Sorting and Searching, Dynamic Programming, Graph Algorithms, Range Queries, Tree Algorithms, Mathematics, String Algorithms, Geometry, Advanced Techniques, Additional Problems
- **Best for**: building a complete CP foundation; every serious competitor does CSES
- Pairs perfectly with the **[Competitive Programmer's Handbook](https://cses.fi/book/book.pdf)** (free, by the same author, Antti Laaksonen)

### CSES completion order
1. Introductory Problems (warmup)
2. Sorting and Searching ⭐
3. Dynamic Programming ⭐
4. Graph Algorithms ⭐
5. Tree Algorithms
6. Range Queries
7. Mathematics
8. String Algorithms
9. Geometry, Advanced Techniques

---

## A2OJ LADDERS (Codeforces difficulty progression)
- Problem ladders grouped by Codeforces rating ranges
- Solve in order to climb steadily
- Great structured alternative to random problem selection
- (A2OJ has had uptime variability; mirror sites and the "CF ladders" on other trackers exist)

---

## CODEFORCES /EDU ⭐ (interactive courses)
- **ITMO Academy** courses: binary search, segment trees (parts 1 & 2), suffix structures, two pointers, DSU, and more
- Interactive with steps + problems
- Free at [codeforces.com/edu](https://codeforces.com/edu)

---

## USACO GUIDE ⭐ (olympiad + CP curriculum)
- [usaco.guide](https://usaco.guide) — phenomenal free structured curriculum
- **Divisions**: Bronze → Silver → Gold → Platinum (mirrors USACO contest levels)
- Covers everything from basics to advanced (segment trees, flows, etc.)
- Excellent module structure with problems from multiple judges

---

## ATCODER EDUCATIONAL DP CONTEST (EDPC) ⭐
- 26 problems (A-Z) covering ALL foundational DP
- A must-do for DP mastery
- At [atcoder.jp](https://atcoder.jp)

## ATCODER LIBRARY PRACTICE CONTEST (ACL)
- Problems exercising advanced data structures (segtree, lazy segtree, DSU, max flow, MCMF, FFT)

---

## CP RESOURCE BOOKS (free)
- **Competitive Programmer's Handbook** (Laaksonen) — free PDF ⭐
- **[CP-Algorithms.com](https://cp-algorithms.com)** — the encyclopedia ⭐

## THE CP LADDER (which to use when)
| Stage              | Resource                              |
|--------------------|---------------------------------------|
| Absolute start     | CSES Introductory + CP Handbook       |
| Building basics    | CSES Sorting/Searching, DP, Graphs    |
| Topic deep-dives   | CF /edu (ITMO Academy)                |
| Olympiad track     | [USACO Guide](https://usaco.guide) (Bronze→Platinum)         |
| DP mastery         | [AtCoder](https://atcoder.jp) EDPC                          |
| Advanced DS        | AtCoder ACL + [CP-Algorithms](https://cp-algorithms.com)           |
| Rating climbing    | A2OJ/CF ladders + live contests       |
