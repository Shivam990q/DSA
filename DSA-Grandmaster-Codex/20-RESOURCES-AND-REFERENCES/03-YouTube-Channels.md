# 📺 YouTube Channels

> *"Free expert teaching, 24/7. Curate wisely; don't fall into tutorial hell."*

---

## BEGINNER-FRIENDLY
- **[CS50](https://cs50.harvard.edu) (Harvard)** — David Malan, legendary intro
- **mycodeschool** — recursion + DS gold (classic)
- **[Abdul Bari](https://www.youtube.com/@abdul_bari)** — algorithm fundamentals, clear visuals
- **[NeetCode](https://neetcode.io)** ⭐ — best interview problem walkthroughs

## INTERVIEW PREP
- **NeetCode** ⭐ — Blind 75 + 150, pattern-based
- **Take U Forward (Striver)** ⭐ — A2Z + SDE sheets (Indian audience)
- **[Tushar Roy](https://www.youtube.com/@tusharroy2525)** — FAANG-style explanations
- **Kunal Kushwaha** — beginner-to-intermediate

## DYNAMIC PROGRAMMING
- **[Aditya Verma](https://www.youtube.com/@AdityaVermaTheProgrammingLord)** ⭐ — best DP + recursion playlist
- **NeetCode** — DP playlist

- **Take U Forward** — complete graph series

## COMPETITIVE PROGRAMMING

## THEORY / ADVANCED

- **[Reducible](https://www.youtube.com/@Reducible)** — beautiful algorithm visualizations
