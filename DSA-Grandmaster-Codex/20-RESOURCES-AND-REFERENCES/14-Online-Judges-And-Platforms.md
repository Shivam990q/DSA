## 🏋️ PRACTICE / LEARNING PLATFORMS

### Interview-focused
- **[LeetCode](https://leetcode.com)** ⭐ — the standard; company tags, contests, study plans
- **[HackerRank](https://www.hackerrank.com)** ⭐ — skill certifications, interview kits, used by many companies for OAs
- **[CodeSignal](https://codesignal.com)** — interview practice + the platform many companies use for OAs
- **[InterviewBit](https://www.interviewbit.com)** — structured interview prep with a track system
- **Coderbyte** — challenges + interview kits
- **[AlgoExpert](https://www.algoexpert.io)** (paid) — 160+ curated with video
- **[Codewars](https://www.codewars.com)** — gamified kata practice
- **Edabit** — bite-sized challenges (beginners)

### Competitive Programming
- **[Codeforces](https://codeforces.com)** ⭐ — the home of CP, rated rounds
- **[AtCoder](https://atcoder.jp)** ⭐ — clean problems, ABC/ARC/AGC
- **[CodeChef](https://www.codechef.com)** — long challenges, starters
- **[CSES](https://cses.fi/problemset/)** ⭐ — structured problem set
- **[TopCoder](https://www.topcoder.com)** — classic SRMs
- **[SPOJ](https://www.spoj.com)** — classic archive
- **[HackerEarth](https://www.hackerearth.com)** — monthly contests + OA platform
- **[DMOJ](https://dmoj.ca)**, **Timus**, **UVa**, **[Kattis](https://open.kattis.com)** (ICPC), **[Library Checker](https://judge.yosupo.jp)** (verify CP code)
- **[Project Euler](https://projecteuler.net)** — math + programming

### Olympiad
- **[USACO](https://www.usaco.org)** (free, monthly) + **[USACO Guide](https://usaco.guide)**
- **[oi.uz](https://oi.uz)** (IOI + national archives)

Knowing the OA (Online Assessment) platform helps you prepare for its format:

| Platform     | Used by (examples)                       | Notes                                   |
|--------------|------------------------------------------|-----------------------------------------|
| **HackerRank** | Many product + service companies        | DSA + sometimes role-specific; certs    |

| **CodinGame / CodinGame Assessment** | Some companies     | Game-like                               |
| **Mettl / Mercer Mettl** | Indian service companies       | Aptitude + coding + proctoring          |
| **AMCAT / Cocubes** | Service companies (employability)  | Aptitude-heavy                          |
| **TCS iON / NQT** | TCS                                   | NQT format                              |
| **HireVue**    | Some companies                          | Video + sometimes coding                |

## 🎯 PREPARING FOR EACH OA TYPE
- **HackerRank OA**: practice on HackerRank itself; expect 1-3 problems, sometimes with hidden test cases. Get HackerRank certifications (free, signal to recruiters).
- **CodeSignal GCA**: 4 questions, ~70 min, scored 600-850. Practice CodeSignal's arcade + their practice GCA.
- **Codility**: practice Codility lessons (they're free); tasks emphasize correctness + performance test cases.
- **HackerEarth**: practice their problems; campus drives often use it.
- **Service company tests (TCS/Infosys/Wipro)**: use [PrepInsta](https://prepinsta.com) + [Faceprep](https://www.faceprep.in) for exact patterns.

## 🗺️ WHICH PLATFORM FOR WHICH GOAL
| Goal                  | Primary platform(s)                   |
|-----------------------|---------------------------------------|
| FAANG interviews      | LeetCode (+ [NeetCode](https://neetcode.io) lists)           |
| Indian product/service| LeetCode + HackerRank + GfG + PrepInsta |
| Competitive programming | Codeforces + AtCoder + CSES         |
| Olympiad              | USACO Guide + Codeforces              |
| OA prep (specific)    | The platform the company uses + its practice mode |
| Mock interviews       | Pramp (free) + interviewing.io (paid) |

---

## THE PRINCIPLE
- For LEARNING: pick LeetCode (interviews) or Codeforces/AtCoder (CP) as primary.
- For OA PREP: practice on the SAME platform the company uses, in its format.
- For MOCKS: do them — they're the bridge from "can solve" to "can perform."
