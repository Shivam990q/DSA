# 🏋️ Practice Platforms

> *"The gym where skill is built. Pick a primary; train daily."*

---

## TIER 1 — MUST USE
- **[Codeforces](https://codeforces.com)** ⭐ — rated rounds, the home of CP
- **[AtCoder](https://atcoder.jp)** ⭐ — cleaner problems, great for observation
- **[LeetCode](https://leetcode.com)** ⭐ — interview prep
- **[CSES Problem Set](https://cses.fi/problemset/)** ⭐ — structured CP curriculum

## TIER 2
- **[CodeChef](https://www.codechef.com)** — long challenges, cook-offs
- **[HackerRank](https://www.hackerrank.com)** — skill paths, beginner-friendly
- **[HackerEarth](https://www.hackerearth.com)** — monthly contests
- **Codeforces Gym** — past ICPC contests
- **[Kattis](https://open.kattis.com)** — ICPC problem archive
- **[Project Euler](https://projecteuler.net)** — math + programming puzzles

## TIER 3 / SPECIALIZED
- **[USACO](https://www.usaco.org)** — olympiad, subtask-style (free)
- **[SPOJ](https://www.spoj.com)** — classic problems
- **[DMOJ](https://dmoj.ca)** — CP-focused judge
- **Timus** — classic algorithmic problems
- **UVa Online Judge** — huge archive
- **Topcoder** — older, classic SRMs

## BY GOAL
| Goal              | Platform                          |
|-------------------|-----------------------------------|
| FAANG interviews  | LeetCode (+ [NeetCode](https://neetcode.io) lists)       |
| Competitive prog  | Codeforces + AtCoder + [CSES](https://cses.fi/problemset/)       |
| Olympiad (IOI)    | [USACO Guide](https://usaco.guide) + [oi.uz](https://oi.uz)               |
| ICPC              | Codeforces Gym + Kattis           |
| Math + algo       | Project Euler                     |
| Indian placements | LeetCode + GFG + Striver sheets   |

## STRUCTURED LISTS / LADDERS
- **NeetCode 150** ⭐
- **Striver A2Z + SDE Sheet** ⭐
- **Blind 75**
- **A2OJ Ladders** (CF difficulty progression)
- **CSES sections**

---

## THE PRINCIPLE
Pick ONE primary platform for your goal. Train daily. Add others only after exhausting the first. Depth over breadth.
