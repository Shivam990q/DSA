## HOW TO WORK THIS LIST
1. Go tier by tier (difficulty increases).
2. For each: attempt 30 min → editorial idea → re-derive → implement → note pattern.
3. Star the hard ones; revisit at 7 and 30 days.
4. After all 50 + EDPC, DP recognition becomes instinct.
