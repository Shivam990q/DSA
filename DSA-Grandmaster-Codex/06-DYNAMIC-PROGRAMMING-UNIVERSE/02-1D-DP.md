# 1️⃣ 1D Dynamic Programming

> *"The simplest DP: a single dimension, a single decision per step."*

---

## I. THE SHAPE

State: `dp[i]` indexed by position i.
Transition: `dp[i]` depends on `dp[i-1]`, `dp[i-2]`, ... (a bounded look-back).

---

## II. THE CANONICAL PROBLEMS

### Climbing Stairs (LC 70)
Ways to climb n stairs taking 1 or 2 steps.
- State: `dp[i]` = ways to reach step i
- Transition: `dp[i] = dp[i-1] + dp[i-2]`
- Base: `dp[0] = dp[1] = 1`
- Answer: `dp[n]`
- **Space optimization**: only need last two → O(1).

```python
def climbStairs(n):
    a, b = 1, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

### House Robber (LC 198)
Max sum, no two adjacent.
- State: `dp[i]` = max sum from houses 0..i
- Transition: `dp[i] = max(dp[i-1], dp[i-2] + nums[i])`
- The choice: skip house i (take dp[i-1]) or rob it (take dp[i-2] + nums[i]).

```python
def rob(nums):
    prev2, prev1 = 0, 0
    for x in nums:
        prev2, prev1 = prev1, max(prev1, prev2 + x)
    return prev1
```

### House Robber II (LC 213, circular)
First and last are adjacent. Run House Robber twice: exclude first, exclude last; take max.

### Min Cost Climbing Stairs (LC 746)
`dp[i] = min(dp[i-1] + cost[i-1], dp[i-2] + cost[i-2])`

### Decode Ways (LC 91)
Count ways to decode digit string into letters (A=1..Z=26).
- `dp[i] = dp[i-1]` (if s[i-1] is 1-9) `+ dp[i-2]` (if s[i-2..i-1] is 10-26)

### Maximum Subarray (LC 53, Kadane)
- `dp[i]` = max sum subarray ending at i = `max(nums[i], dp[i-1] + nums[i])`
- Answer = max over all i.

### Maximum Product Subarray (LC 152)
Track BOTH max and min ending at i (negatives flip signs):
```python
def maxProduct(nums):
    res = mx = mn = nums[0]
    for x in nums[1:]:
        if x < 0: mx, mn = mn, mx
        mx = max(x, mx * x)
        mn = min(x, mn * x)
        res = max(res, mx)
    return res
```

### Word Break (LC 139)
`dp[i]` = can s[0..i-1] be segmented?
`dp[i] = OR over j < i: dp[j] AND s[j..i-1] in dict`

### Number of Dice Rolls With Target Sum (LC 1155)
1D over target sum, iterate dice.

---

## III. THE 1D DP RECIPE

1. Define `dp[i]` = answer for prefix/state ending at i.
2. Figure out transition: how does dp[i] use earlier dp values?
3. Identify base case(s).
4. Iterate i from small to large.
5. Answer is usually dp[n] or max/min over all dp[i].
6. Optimize space if look-back is bounded.

---

## IV. SPACE OPTIMIZATION

If `dp[i]` only uses `dp[i-1]` and `dp[i-2]`:
```python
# Instead of array, keep two variables
prev2, prev1 = base0, base1
for i in range(2, n+1):
    cur = f(prev1, prev2)
    prev2, prev1 = prev1, cur
return prev1
```

Reduces O(n) space to O(1).
