# 📖 Meta's Hardest Interview Problems

> *"Meta interviews are fast and direct — often 2 problems in 35 minutes. Speed + clarity wins."*

---

## I. THE META STYLE
- Fast-paced: ~2 problems in 35 minutes
- Favors trees, graphs, BFS/DFS, and clean optimization
- Direct problem statements
- Values: move fast, impact, clean code

---

## II. HARD PROBLEMS FREQUENTLY ASSOCIATED WITH META

### 1. Minimum Remove to Make Valid Parentheses (LC 1249)
- Stack / two-pass to mark removable parens.
- **Lesson**: clean string manipulation.

### 2. Subarray Sum Equals K (LC 560)
- Prefix sum + hash map of counts.
- **Lesson**: prefix sum frequency.

### 3. Valid Palindrome II (LC 680)
- Two pointers, allow one deletion.
- **Lesson**: greedy with one "skip."

### 4. Binary Tree Vertical Order Traversal (LC 314)
- BFS + column indexing + hash map.
- **Lesson**: coordinate bucketing on trees.

### 5. Lowest Common Ancestor (LC 236)
- Recursive DFS returning found-status.
- **Lesson**: classic tree recursion.

### 6. Serialize and Deserialize Binary Tree (LC 297)
- Preorder with null markers.
- **Lesson**: encoding tree structure.

### 7. Alien Dictionary (LC 269)
- Build graph of letter precedence → topological sort.
- **Lesson**: derive a graph from ordering constraints.

### 8. Random Pick with Weight (LC 528)
- Prefix sums + binary search.
- **Lesson**: weighted sampling via prefix + BS.

### 9. Continuous Subarray Sum (LC 523)
- Prefix sum mod k + hash map.
- **Lesson**: modular prefix sums.

### 10. Kth Largest Element (LC 215)
- Quickselect (O(n) avg) or min-heap of size k.
- **Lesson**: selection without full sort.

---

## III. THE META-SPECIFIC SKILLS
- **Speed**: solve Mediums in <15 min (2 in 35)
- **Cleanliness**: bug-free first try
- **Trees & graphs**: heavily favored
- **Communication**: concise, no rambling

---

## IV. PREPARATION
- Meta-tagged problems (LC Premium)
- Practice 2-problems-in-35-min drills
- Speed + accuracy focus
- Trees, graphs, prefix sums, hash maps

---

## V. THE BAR
Meta wants engineers who ship fast and correctly. Demonstrate velocity WITHOUT sacrificing correctness. Practice until Mediums are reflexive.
