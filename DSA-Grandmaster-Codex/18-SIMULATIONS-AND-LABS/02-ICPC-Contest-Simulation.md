# 🏆 ICPC Contest Simulation

> *"Five hours. One computer. Three minds. Thirteen problems. This is war."*

---

## I. THE FORMAT
- 3 students, 1 computer
- 5 hours
- 8-13 problems
- Ranked by problems solved, then total time (with penalties for wrong submissions)

---

## II. SETUP THE SIMULATION
- Gather a team of 3 (or simulate solo with 5h timer)
- Pick a past ICPC regional/world final from [Codeforces](https://codeforces.com) gym or [Kattis](https://open.kattis.com)
- One computer rule (forces collaboration)
- Print problems (ICPC tradition)
- No internet except the judge

---

## III. THE TEAM STRATEGY
```
Min 0-15:   ALL THREE read problems (split: each reads ~4)
Min 15:     Quick sync — rank problems by difficulty
Min 15-60:  Implementer codes easiest 2-3; others think on mediums
Min 60-180: Rotate computer; pair-program hard problems
Min 180-280: Push on remaining; debug; one verifies while one codes
Min 280-300: Final submissions; don't waste penalty on untested code
```

---

## IV. ROLES
- **Implementer**: fastest, cleanest coder; types most solutions
- **Algorithmist**: strongest theory; cracks the hard problems
- **Captain**: tracks scoreboard, time, decides what to attempt next

Rotate based on strengths per problem.

---

## V. THE PENALTY SYSTEM
- Each solved problem: time of submission (from contest start) added
- Each wrong submission: +20 min penalty (only counts if eventually solved)
- → Don't spam submissions. Test locally first.

---

## VI. THE COMPUTER-SHARING DISCIPLINE
With one computer:
- Think on PAPER, code only when solution is clear
- "Coding time" is precious — don't think at the keyboard
- Pre-write solutions on paper, transcribe quickly

---

## VII. POST-SIMULATION DEBRIEF
- Which problems did we solve? Miss?
- Did we read all problems early?
- Did we waste computer time thinking?
- Did we waste penalty on untested submissions?
- What pattern did we miss?
- Upsolve ALL unsolved over the next 2 days.

---

## VIII. PRACTICE CADENCE
- 1 team virtual ICPC per week (5h)
- Upsolve over the following week
- Build team chemistry (who's best at what)
