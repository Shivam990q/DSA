# 🥉 Implementation Trial Bank

> *"If you can't implement it from scratch under time pressure, you don't own it."*

For each level, implement the listed structures/algorithms from memory, no references, within the time limit.

---

- [ ] Binary search

---

## THE TRIAL PROTOCOL
1. Pick the items for your level.
2. Set the time limit.
3. No references, no autocomplete.
4. Implement, then verify against a known-correct implementation.
5. Bug-free first try = PASS. Otherwise, drill that item.

---

## RE-VERIFICATION
Re-run your level's implementation trial monthly. Skills rust. Re-cement.
