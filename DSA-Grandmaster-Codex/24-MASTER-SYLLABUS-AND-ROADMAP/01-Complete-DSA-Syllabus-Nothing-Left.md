## 🔑 THE 13 UNIVERSAL PATTERNS (the real "syllabus")

Memorize these triggers — 90%+ of problems are one of these (or a combination):

| Pattern | Use when |
|---------|----------|
| Two Pointers / Sliding Window | array/string subarray/substring constraint |
| Prefix/Suffix + Difference Array | range updates, static range queries |
| Monotonic Stack/Queue | next/prev greater-smaller, window min/max |
| Binary Search on Answer | monotonic feasibility, min-max optimization |
| Meet-in-the-Middle | n ≤ 40, exponential but halvable |
| Coordinate Compression | large value ranges, sparse data |
| Offline + Mo's / DSU Rollback | batch queries, static/semi-static data |
| Rerooting DP / Euler Tour | tree queries from every node |
| CHT / Li Chao / Slope Trick | DP with linear/quadratic transitions |
| Digit DP / Bitmask DP | constraints on digits / small-state masks |
| SOS DP | subset/superset sums, bitwise transitions |
| Parallel Binary Search | many monotonic queries over a range |
| Constructive / Ad-hoc | no standard structure → observation + building |

Full treatment: [`../13-INTERVIEW-DSA-UNIVERSE/02-The-25-Patterns.md`](../13-INTERVIEW-DSA-UNIVERSE/02-The-25-Patterns.md) and [`../16-MENTAL-MODELS-AND-FRAMEWORKS/`](../16-MENTAL-MODELS-AND-FRAMEWORKS/).

## 🚫 NOT IN THE SYLLABUS (stop wasting time)

- ❌ Solving 1000+ random problems (150 curated > 1000 random)
- ❌ Memorizing solutions instead of patterns
- ❌ Obscure DS for interviews (B-Tree, R-Tree, Rope, Skip List internals) — concept only
- ❌ Platform-specific tricks
- ❌ Over-optimizing beyond what constraints require
- ❌ "Har problem type yaad karna" — remember PATTERNS, not problems

## ✅ THE "NOTHING LEFT" COMPLETION CHECKLIST

DSA is complete for you when ALL six are true:

1. [ ] You can write any **core DS from scratch** (no library) in <10 min: stack, heap, BST, Trie, DSU, segment tree, graph traversal.
2. [ ] You **identify the pattern** of a new problem in 3-5 min.
3. [ ] You can **derive** a DP state + transition (+ optimization if needed) yourself.
4. [ ] You convert **brute → optimal** consistently, solving Medium in <30 min, Hard in <45.
5. [ ] You can **prove** time/space bounds (amortized, Master theorem, potential method).
6. [ ] You handle **edge cases automatically**: n=1, duplicates, negatives, overflow, recursion depth, empty input.

If all six are checked → **DSA Universe complete.** Move energy to system design + behavioral (see [`03-FAANG-Reality-Check.md`](./03-FAANG-Reality-Check.md)).
