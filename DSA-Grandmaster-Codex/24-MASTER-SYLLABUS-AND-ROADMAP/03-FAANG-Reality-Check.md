# 🎯 FAANG / MAANG Reality Check — DSA is Necessary, Not Sufficient

> *"Sirf DSA karke FAANG jaaoge → reject. DSA + full package → offer. Ye honest breakdown padho taaki tum sahi cheez pe time lagao."*

---

## ⚖️ THE HONEST WEIGHTAGE

DSA is roughly **60-70%** of the evaluation (the coding rounds). The other **30-40%** is everything else — and that's where most well-prepared coders actually fail.

| Round | Weight | What it tests | Covered in DSA syllabus? |
|-------|--------|---------------|--------------------------|
| Coding (2-4 rounds) | ~60-70% | Arrays, strings, trees, graphs, DP, heaps, tries, backtracking | ✅ Yes — Tier 1 + 80% Tier 2 |
| System Design (L4+) | ~20% | LLD (OOP, design patterns) + HLD (scalability, DBs, caching, load balancers) | ❌ Separate skill |
| Behavioral / LP / Googleyness | ~15% | STAR stories, ownership, conflict, ambiguity | ❌ Separate skill |
| CS Fundamentals | ~5-10% | OS, DBMS, networks, OOP, multithreading (role-dependent) | ❌ Separate skill |

**Bottom line**: DSA gets you THROUGH the coding rounds. It does not get you the offer by itself.

---

## ✅ DSA: how much is "enough" for FAANG?

| Topic group | FAANG frequency | Verdict |
|-------------|-----------------|---------|
| Arrays, strings, hashing, two pointers, sliding window | 🔥 ~90% of rounds | **Must master** |
| Trees/BST, heaps, tries, graphs (BFS/DFS/Dijkstra/MST/topo) | 🔥 ~80% | **Must master** |
| DP (1D/2D, knapsack, LCS/LIS, grid), backtracking | 🔥 ~70% | **Must master** |
| Monotonic stack, binary search on answer, greedy, intervals | ✅ ~40% | **High ROI** |
| Segment tree (lazy), HLD, FFT/NTT, flow, blossom, advanced geometry | ❌ ~0-2% | **Skip unless CP** |
| Advanced math (Möbius, FWHT, Berlekamp, Gaussian elim) | ❌ ~0% | **Skip for interviews** |

> **FAANG DSA sweet spot = LeetCode Medium + ~20% Hard.** FAANG does NOT ask ICPC-level data structures, obscure math, or CP optimization tricks. So: **Tier 1 + 80% of Tier 2 = 100% of FAANG coding-round coverage.** Tier 3 is for competitive programming, not interviews.

## 💡 THE FINAL VERDICT

| If you do | Result |
|-----------|--------|
| Only DSA + 1000 random problems | ❌ Reject (communication/design missing) |
| DSA patterns + 150 curated + clean code + mocks | ✅ Coding rounds clear |
| DSA + System Design + Behavioral + Mocks | 🏆 FAANG/MAANG offer |

**Truth**: clearing DSA is *necessary* but not *sufficient*. Pattern recognition + communication + system design + behavioral = offer.

---

## 🚀 WHAT TO DO WITH THIS

- If you're early: focus 100% on DSA (Steps 1-20 in [`02-The-Sequenced-Path.md`](./02-The-Sequenced-Path.md)). Don't touch system design yet.
- Once you solve Medium reliably: start adding system design + behavioral in parallel (Step 29).
- 4-6 weeks before interviews: mocks become the priority. Speed and calm beat knowing one more obscure algorithm.
